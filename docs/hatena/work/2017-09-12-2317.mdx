### いざ起動

USBハブと、wifiドングルと、HDMIケーブルと、キーボードをつなげて、最後に電源ケーブルを挿すと、起動。

キーボードが認識してくれれば、普通に入力できる。  
初回のユーザーはpi、でログインパスワードはraspberryでログインできる。

### キーボードレイアウトを変更する。

以下のコマンドでラズパイのコンフィグ画面を開く。

```
$ sudo raspi-config
```

キーボードレイアウトを自分の環境に変更しておこう。  
そのままで問題なければいいんだけれども、パイプ「|」とか打てなくて面倒だったりする。

#### 事前の確認

**ネットワークカードであるwifiドングルが認識していること**

```
$ lsusb
Bus 001 Device 004: ID 0411:01ee BUFFALO INC. (formerly MelCo., Inc.) WLI-UC-GNM2 Wireless LAN Adapter [Ralink RT3070]
Bus 001 Device 005: ID 05ac:0251 Apple, Inc. 
Bus 001 Device 003: ID 05ac:1006 Apple, Inc. Hub in Aluminum Keyboard
Bus 001 Device 002: ID 05e3:0608 Genesys Logic, Inc. Hub
Bus 001 Device 001: ID 1d6b:0002 Linux Foundation 2.0 root hub
```

Wireless Lan Adapter〜みたいなのがあるからOK！

**同じくwifiドングルのデバイス名があること**

```
pi@raspberrypi:~ $ ifconfig
wlan0 Link encap:Ethernet HWaddr 消した
          UP BROADCAST MULTICAST MTU:1500 Metric:1
          RX packets:0 errors:0 dropped:0 overruns:0 frame:0
          TX packets:0 errors:0 dropped:0 overruns:0 carrier:0
          collisions:0 txqueuelen:1000
          RX bytes:0 (0.0 B) TX bytes:0 (0.0 B)
```

wlan0っていうのがDebian系の無線のデバイス名みたい！ あればOK！
