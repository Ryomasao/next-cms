この記事は、

これが必要みたい。

```
sudo modprobe bcm2835-v4l2
```

「mjpg\_streamer」があるディレクトリに移動。

```
cd /home/pi/mjpg-streamer/mjpg-streamer-experimental
```

何も考えずに実行する。

```
sudo ./mjpg_streamer -i "./input_uvc.so -f 10 -r 320x240 -d /dev/video0 -y -n" -o "./output_http.so -w ./www -p 8080"
```
