なので、RaspBerryPi用にバッテリーを購入しました。

### 問題発生

問題というか、冷静に考えれば当たり前のことなんですが、

![f:id:konoemario:20170309212347j:plain:w300](https://cdn-ak.f.st-hatena.com/images/fotolife/k/konoemario/20170309/20170309212347.jpg "f:id:konoemario:20170309212347j:plain:w300")

↑に↓これらのパーツを積む必要があります。

![f:id:konoemario:20170306221652j:plain:w300](https://cdn-ak.f.st-hatena.com/images/fotolife/k/konoemario/20170306/20170306221652.jpg "f:id:konoemario:20170306221652j:plain:w300")

パーツ達

```
・電池
・ブレッドボード
・RaspBerryPi
・バッテリー
```

つまり、みりんちゃんが座っているスペースなぞ、取っている余裕はないのです。  
いろいろと試行錯誤した結果、蚊取り線香の箱に、いろいろと乗せて、キャタピラカーがそれを引っ張るという形に落ち着きました。
